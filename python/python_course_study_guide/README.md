# Python Course Study Guide

## Overview
This study guide is based on the "Curso em Vídeo - Python" playlist by Gustavo Guanabara, a comprehensive introduction to Python 3 for beginners. It covers fundamentals, control structures, and compound data types, progressing from basic concepts to advanced topics.

## Structure
- **Topics**: Individual Markdown files for each major concept, with theoretical explanations, code examples, and syntax summaries.
- **Exercises**: A centralized file with all practical challenges, organized by module and difficulty.

## Learning Path
Follow the numerical order in the `topics/` folder, then practice with the exercises in `exercises/`.
