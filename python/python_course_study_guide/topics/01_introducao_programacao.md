# Introdução à Programação e Python

## Explicação Teórica
Programação é o processo de criar instruções para um computador executar tarefas. Python é uma linguagem de programação de alto nível, interpretada, fácil de aprender e amplamente usada em desenvolvimento web, ciência de dados, automação e mais. Foi criada por Guido van Rossum em 1991 e é conhecida por sua sintaxe limpa e legibilidade.

Por que aprender Python?
- Sintaxe simples e intuitiva.
- Comunidade ativa e vasto ecossistema de bibliotecas.
- Versátil: scripts, aplicações web, IA, jogos.
- Gratuita e multiplataforma.

## Exemplos de Código
```python
# Primeiro programa em Python
print("Olá, mundo!")
```

```python
# Calculadora simples
numero1 = 10
numero2 = 5
soma = numero1 + numero2
print(f"A soma é {soma}")
```

## Resumo de Sintaxe
- `print()`: Exibe texto ou valores na tela.
- Variáveis: Atribuídas com `=`, sem declaração de tipo.
- Comentários: Iniciam com `#`.
