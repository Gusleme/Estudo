# Primeiros Comandos em Python

## Explicação Teórica
Comandos básicos incluem entrada/saída e manipulação simples. Use `print()` para saída e `input()` para entrada do usuário. Python executa linha por linha.

## Exemplos de Código
```python
# Saudação personalizada
nome = input("Digite seu nome: ")
print(f"Olá, {nome}!")
```

```python
# Conversão simples
idade = int(input("Digite sua idade: "))
print(f"Você tem {idade} anos.")
```

## Resumo de Sintaxe
- `input(prompt)`: Lê entrada do usuário como string.
- `int()`, `float()`, `str()`: Conversão de tipos.
