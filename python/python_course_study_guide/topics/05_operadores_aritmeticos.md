# Operadores Aritméticos

## Explicação Teórica
Operadores realizam cálculos: + (soma), - (subtração), * (multiplicação), / (divisão), // (divisão inteira), % (resto), ** (potência). Ordem de precedência: parênteses, potência, multiplicação/divisão, soma/subtração.

## Exemplos de Código
```python
# Cálculos básicos
a = 10
b = 3
print(a + b)   # 13
print(a ** b)  # 1000
print(a % b)   # 1
```

```python
# Área de um retângulo
largura = float(input("Largura: "))
altura = float(input("Altura: "))
area = largura * altura
print(f"Área: {area}")
```

## Resumo de Sintaxe
- `+`, `-`, `*`, `/`, `//`, `%`, `**`.
- Precedência: PEMDAS (Parênteses, Expoentes, Multiplicação/Divisão, Adição/Subtração).
