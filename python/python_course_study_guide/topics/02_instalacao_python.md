# Instalação do Python e Ferramentas

## Explicação Teórica
Para programar em Python, instale o interpretador oficial do site python.org. Use o IDLE (ambiente integrado) para testes simples ou PyCharm para projetos maiores. No celular, use QPython3.

Passos básicos:
1. Baixe o Python 3.x.
2. Execute o instalador.
3. Verifique com `python --version` no terminal.

## Exemplos de Código
Nenhum código novo; foque na instalação.

## Resumo de Sintaxe
- Comando no terminal: `python arquivo.py` para executar um script.
